# Grama-Sanjeevini 🌿💊

Grama-Sanjeevini is an Android-based rural healthcare application developed to improve medicine accessibility in villages and semi-rural areas through real-time pharmacy connectivity.

## 📱 Features

- 🔍 Medicine Search
- 🎤 Voice-based Search
- 🚨 Emergency Medicine Section
- 🏥 Nearby Pharmacy Listings
- 📦 Real-time Inventory Management
- 👨‍⚕️ Pharmacist Dashboard
- 🔐 Firebase Authentication
- ☁️ Firebase Firestore Integration
- 🤖 AI-assisted Medicine Search
- 🌐 Lightweight and Rural-Friendly UI

---

## 🛠️ Technologies Used

- Kotlin
- Android Studio
- Firebase Authentication
- Firebase Firestore
- RecyclerView
- XML Layouts
- SharedPreferences
- Git & GitHub

---

## 📌 Project Objective

The objective of Grama-Sanjeevini is to help rural users quickly locate medicines and nearby pharmacies while reducing unnecessary travel and improving emergency healthcare accessibility.
